PROGRAMAÇÃO



SINTAXE

Principais sintaxes da linguagem de programação Go (GoLang)

1. DECLARAÇÃO DE VARIÁVEIS: Go utiliza a palavra-chave `var` para declarar variáveis. Exemplo: `var nome string`.

2. ATRIBUIÇÃO DE VALORES: A atribuição é feita utilizando `=`. Exemplo: `nome = “João”`.

3. DECLARAÇÃO E INICIALIZAÇÃO DE VARIÁVEIS: Você pode declarar e atribuir valores simultaneamente: `idade := 25`.

4. TIPOS DE DADOS BÁSICOS: Go possui tipos como `int`, `float64`, `string`, `bool`, etc.

5. ESTRUTURAS DE CONTROLE:
- `if` para estruturas condicionais.
- `for` para loops.
- `switch` para seleção múltipla.

6. FUNÇÕES: Declaração de função utiliza `func`. Exemplo:

```go
Func saudacao(nome string) {
Fmt.Println(“Olá, “ + nome)
}

7. PONTEIROS: Go permite o uso de ponteiros para acessar e modificar valores na memória.

8. STRUCTS: Você pode criar tipos de dados personalizados usando `struct`. Exemplo:

```go
Type Pessoa struct {
Nome string
Idade int
}

9. SLICES E ARRAYS: Go oferece slices dinâmicos e arrays estáticos para armazenar coleções de elementos.

10. PACOTES E IMPORTS: O código Go é organizado em pacotes. Usamos `import` para usar código de outros pacotes.

FUNÇÕES

Principais funções e recursos da linguagem de programação Go (GoLang):

1. fmt Package:
- `Println()`: Imprime no console com uma nova linha.
- `Printf()`: Permite formatação de strings para impressão.
- `Scanln()`: Lê entrada do usuário a partir do console.

2. strings Package:
- `len()`: Retorna o comprimento de uma string.
- `Concat()`: Concatena strings.

3. strconv Package:
- `Atoi()`: Converte uma string para um inteiro.
- `Itoa()`: Converte um inteiro para uma string.

4. math Package:
- Funções matemáticas, como `Sqrt()`, `Abs()`, `Sin()`, `Cos()`, etc.

5. time Package:
- `Now()`: Retorna a hora atual.
- `Sleep()`: Pausa a execução por um período de tempo.

6. os Package:
- `Create()`: Cria um novo arquivo.
- `Open()`: Abre um arquivo existente.
- `Remove()`: Remove um arquivo.

PROGRAMAS EXEMPLO

1-Vetor (Slice):

package main

Import “fmt”

Func main() {
Numbers := []int{1, 2, 3, 4, 5}
Fmt.Println(“Original slice:”, numbers)

// Adicionar um elemento ao slice
Numbers = append(numbers, 6)
Fmt.Println(“Slice after appending:”, numbers)

// Remover o primeiro elemento do slice
Numbers = numbers[1:]
Fmt.Println(“Slice after removing first element:”, numbers)

// Acessar elementos individuais
Fmt.Println(“First element:”, numbers[0])
Fmt.Println(“Last element:”, numbers[len(numbers)-1])
}




2-Criptografia:

package main

Import (
“crypto/aes”
“crypto/cipher”
“crypto/rand”
“fmt”
“io”
)

Func main() {
Key := []byte(“0123456789abcdef”) // Chave de 16 bytes
Plaintext := []byte(“Hello, encryption!”)

Block, err := aes.NewCipher(key)
If err != nil {
Fmt.Println(“Error:”, err)
Return
}

Ciphertext := make([]byte, aes.BlockSize+len(plaintext))
Iv := ciphertext[:aes.BlockSize]
If _, err := io.ReadFull(rand.Reader, iv); err != nil {
Fmt.Println(“Error:”, err)
Return
}

Mode := cipher.NewCBCEncrypter(block, iv)
Mode.CryptBlocks(ciphertext[aes.BlockSize:], plaintext)

Fmt.Println(“Ciphertext:”, ciphertext)
}




3- Matriz:
package main

Import “fmt”

Func main() {
Rows, cols := 3, 3
Matrix := make([][]int, rows)

For i := range matrix {
Matrix[i] = make([]int, cols)
}

// Preencher a matriz
Value := 1
For i := 0; i < rows; i++ {
For j := 0; j < cols; j++ {
Matrix[i][j] = value
Value++
}
}

// Imprimir a matriz
Fmt.Println(“Matrix:”)
For i := 0; i < rows; i++ {
Fmt.Println(matrix[i])
}
}
